Логирование в Qt проектах. В режиме release записывает в файл события по уровням qInfo, qWarning, qCritical и qFatal. В режиме debug добавляется вывод qDebug и сообщения дублируются в консоль. Файл логирования пишется в каталоге с названием программы в домашней папке.
